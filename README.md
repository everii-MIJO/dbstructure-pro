# DBStructure *Pro*

> **A modern, interactive database schema visualizer and collaboration tool.**

DBStructure *Pro* bridges the gap between database architects and development teams. It transforms static DBML definitions into a dynamic, interactive canvas, making complex relational databases easy to navigate, understand, and refine.

Designed with elegance and speed in mind, it provides a real-time overview of your data architecture while offering built-in tools for team collaboration and structural auditing.

---

## ⚡ Core Capabilities

* **Interactive Canvas:** Seamlessly pan, zoom, and drag tables across an infinite canvas. Your layout preferences are automatically saved.
* **Smart Relationships:** Instantly visualize Primary Keys (PK) and Foreign Keys (FK) with distinct, beautiful styling and intelligent connection routing.
* **Instant Search:** Navigate massive schemas effortlessly. Find any table or column instantly with the built-in spotlight search.
* **Team Collaboration (Notizen):** A built-in task system allowing co-workers to propose structural changes (like missing PKs/FKs). Includes a secure Admin Mode for reviewing and resolving tasks.
* **Elegant UI:** A meticulously crafted interface featuring smooth animations, crisp typography, and seamless Light/Dark mode support.

## 🎯 Why DBStructure Pro?

Understanding a database shouldn't require reading thousands of lines of code. DBStructure *Pro* is built to:
1. **Accelerate Onboarding:** Help new developers understand the data model in minutes, not days.
2. **Ensure Data Integrity:** Visually spot missing relationships or orphaned tables.
3. **Streamline Communication:** Stop sending screenshots. Use the built-in notes system to track structural updates directly alongside the schema.
